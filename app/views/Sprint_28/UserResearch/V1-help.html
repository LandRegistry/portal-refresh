View My Applications

View My Applications displays all of your applications submitted to the HM Land Registry Portal. This guide helps to explain what the features on the page do.


Filter

Filter by application type

Use this to display applications by their type. You can filter by:

  • Register update (an application that changes an existing registered title)
  • Register create (an application to make a new registered title)
  • Register query (an enquiry or search regarding information held at HM Land Registry)

Filter by application progress

Use this to display applications by their current progress status. You can filter by:

  • Received
  • Delayed
  • Cancelled
  • Completed

Note:

Received results include applications with a progress status of:

  • Received: Priority Protected
  • Received: Priority Protected – Awaiting Processing
  • Received: Priority Protected – Set-up activity underway
  • In progress

Applications that are displayed without a progress status will also be shown within the results of this filter option.

Delayed results include applications with a progress status of:

  • Delayed
  • Delayed Prior Pending Application(s) or Official Search(es)
  • Delayed - Extension of Time Granted

Search

Search can locate applications by:

  • Title number
  • HMLR reference
  • Customer reference

The result will show applications that contain the character or string of characters entered as the search term.

Wildcard searching is available. Wildcards allow you to search all of the information that begin with, end with, or include the characters that you specify. The wildcard character is an asterisk (*).

For example, if you search for an application, by putting DN1* it will return all results that begin with DN1.


Sort

Use sort to order your list of applications. You can sort by:

  • Title number
  • Customer reference
  • Received by HMLR
  • Estimated completion date

Initial Application Information (collapsed row)

Title number
Shows the title number for the application (if available). For Land Charges Registration View and Land Charges Office Copy applications it shows the Land Charges Registration Number.

Customer reference
This is the reference provided by you when the application was submitted.

Application type
The type of application you have lodged (for example, Transfer of Part, Official Search of Whole with Priority).

Received by HMLR
This is the date your application was received by HM Land Registry.

Application progress
The high-level progress status of the application (for example, Received, Delayed, Completed or Cancelled).

Note:
Received results include applications with a progress status of:

  • Received: Priority Protected
  • Received: Priority Protected – Awaiting Processing
  • Received: Priority Protected – Set-up activity underway
  • In progress

Applications that are displayed without a progress status will also be shown within the results of this filter option.

Delayed results include applications with a progress status of:

  • Delayed
  • Delayed Prior Pending Application(s) or Official Search(es)
  • Delayed - Extension of Time Granted

Estimated completion date
This shows a projected date for completion of the application, based on average completion time data on the day the application is received. It will change if an application is delayed by correspondence. This date is only displayed for applications that update or create a Register.

Note:
These dates are supplied as estimates only.

Show / hide details (+)
Expands the row so more information about your application can be viewed.


Additional Application Information (expanded row)

Application Progress - details
This shows the progress of an application by the following stages:


Received: Priority Protected

The application has been received and added to our Day List. Its priority has been protected, but no processing activity has taken place.


Received: Priority Protected – Awaiting Processing

This is shown when an application is being held on an office’s backlog list to await processing. No processing activity has yet taken place. The application will normally be dealt with in accordance with its priority order.


Received: Priority Protected – Set-up activity underway

This is shown when the parent title is a developer title and there are setting up activities that need to be completed before applications can proceed.


In Progress

Processing activities are taking place. There are no known events or activities causing the application to be delayed at this time.


Delayed Prior Pending Applications(s) or Official Search(es)

The application has prior pending application(s) or official search(es) with priority on our Day List.


Delayed

Events or activities are causing progress of the application to be delayed.


Delayed - Extension of Time Granted

This is shown when a request for additional time to reply to one or more requisition points has been approved. The amount of additional time allowed can vary and is at the discretion of the Chief Land Registrar based upon the individual circumstances. Subject to approval, a minimum of 10 working days is normally granted in the first instance, while subsequent requests will be treated on their merits. Requests for extension of time can be submitted via the ‘Still Need To Contact Us’ link located within the Application Enquiry service.

HMLR Reference
The unique reference HM Land Registry allocates to an application.

Priority Period
Shown for Official Search with Priority applications only. The full period during which any application protected by a search retains its priority.

Parent Title(s)
This is the existing title or titles out of which a Transfer of Part or New Lease application has been granted.

Correspondence
Shows notices, requisitions and some letters issued by HM Land Registry connected with an application. You can only view the content and download correspondence that is issued to you personally. Correspondence can include:


Letters - Letter issued on [date]

This is shown when HM Land Registry issues a letter in respect of the application.


Notice (Objection):

Notice(s) issued on [date] Expires on [date]

This is shown when HM Land Registry issues a Notice in respect of the application. HM Land Registry serves notices to interested parties to help prevent wrongful registration. A copy notice is not sent to the applicant. Notices inform the recipient that HM Land Registry has received an application, or is otherwise proposing an alteration that may adversely affect their interest. These allow a time period for objection, but also invite the addressee to consent to the application in order to shorten the period of delay.


Notice (Information):

Notice(s) issued on [date] Expires on [None]

This is shown when HM Land Registry issues a Notice for information purposes only. This type of Notice does not have an expiry date and does not require a response.


Requisition(s) issued on [date]

This is shown when HM Land Registry issues a requisition asking for additional information in support of the application. A requisition will not be deemed to have been satisfied until all outstanding points raised have been fully replied to.


Warning of Cancellation issued on [date]

This is shown when HM Land Registry issues a warning that the application will be cancelled unless a full response to a requisition is received.


Files
Shows the documentation electronically issued by HM Land Registry to the applicant on completion of the application. These can be viewed and downloaded.

Show / hide details (-)
Collapses the row so less information about your application is displayed.
